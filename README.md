Nama Aplikasi:
BudgetIn

Deskripsi Singkat Aplikasi:
BudgetIn adalah aplikasi manajemen keuangan pribadi yang membantu pengguna mencatat pemasukan, pengeluaran harian, serta mengelola target finansial (wishlist). Aplikasi ini memudahkan pengguna memonitor kondisi keuangan, melihat statistik pengeluaran, dan menjaga anggaran agar tetap terkontrol.

Fitur yang Akan Ada di Aplikasi
- Login dengan Akun Baru atau Google: Autentikasi digunakan untuk menyimpan data secara aman, mempermudah akses tanpa registrasi manual, serta mendukung sinkronisasi dan pengembangan fitur ke depan.
- Tambah Pengeluaran: Mencatat pengeluaran harian beserta kategori dan detailnya.
- Tambah Pemasukan: Mencatat pemasukan dengan kategori yang tersedia.
- Membuat Wishlist / Target Keuangan: Menambah wishlist dengan gambar, nominal target, dan tujuan yang ingin dicapai.
- Melihat Statistik Pengeluaran: Menampilkan ringkasan grafik untuk memonitor pola dan total pengeluaran.
- Edit Profil: Mengubah informasi akun seperti nama dan foto profil.

Deskripsi Project:
Proyek ini bertujuan untuk membuat aplikasi pencatatan keuangan sederhana yang memungkinkan pengguna mencatat transaksi secara digital. Aplikasi dikembangkan menggunakan Flutter dan ditujukan untuk perangkat Android. Melalui antarmuka yang intuitif, pengguna dapat menambahkan pemasukan atau pengeluaran, melihat ringkasan keuangan, mengatur kategori, membuat wishlist, serta mengelola profil akun.
